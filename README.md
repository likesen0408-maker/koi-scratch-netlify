# Koi Club 刮刮乐本地测试包 v5

这个包用于本地测试，不消耗 Netlify 免费部署次数。

## 启动方法

1. 解压本包。
2. 双击 `start.bat`。
3. 电脑浏览器打开：
   - 客户页面：http://localhost:8787/
   - 后台：http://localhost:8787/admin

## 手机测试

手机和电脑连接同一个 Wi-Fi，然后用电脑 IPv4 打开，例如：

- 客户页面：http://192.168.1.184:8787/
- 后台：http://192.168.1.184:8787/admin

## 内置测试码

- TEST188：18.8 档，可刮 5 张
- TEST288：28.8 档，可刮 5 张

也可以进后台自己生成兑换码。
